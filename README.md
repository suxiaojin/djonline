# djonline
django在线学习网站
简介

djonline是一个基于Django框架实现的在线学习网站，包含了基本的前端功能模块和完善的后台管理系统。特点简单概述如下：
基于Django实现，使用uWSGI和Nginx运行，Python2.7 + Django 1.10.1完美运行
使用MySQL存储数据
基于xadmins对Django Admin进行了深度优化，拥有更好的后台交互和展示
系统集成了UEditor富文本编辑器
完善了分页功能
